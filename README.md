# Controller-EV3
iOS Controller App for Mindstorm EV3 robot (Swift 3.0)

Librairies/API:
  * Joystick : CDJoystick (https://github.com/Coledunsby/CDJoystick)
  * WebSocket : Socket.IO (https://github.com/socketio/socket.io)
